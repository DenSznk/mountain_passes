                Mountain Pass
Выполнил студет курса SkillFactory - PDEV

Данный проект был создан на основе Джанго РЕСТ фреймворк.
Реализует АПИ для получения и отправки информации о состоянии горного
перевала в разное время года. Предусмотрена возможность регистрации пользователя,
создание объектов в базе данных.
Код покрыт тестами для запуска тестов рекомендую использовать готовые фикстуры командой
python manage.py loaddata fixtures/model_name.json --app app.model_name
Используйте подключение всех установленных библиотек используйте команду
pip install -r requirements.txt
Для установки секретного ключа и информации для базы данных используйте
.env_example как пример

This project was created based on the Django REST framework
Implements an API for receiving and sending information about the state of a mountain
pass at different times of the year. Registering a user and creating objects in the database is possible.
The code is covered with tests to run tests, I recommend using ready-made fixtures with the command
python manage.py
loaddata fixtures/model_name.json --app app.model_name
Use the connection of all installed libraries use the command pip install -r requirements.txt
to set the private key and information for the database use .env_example as an example
