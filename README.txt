                Mountain Pass
Выполнил студет курса SkillFactory - PDEV

Данный проект был создан на основе Джанго РЕСТ фреймворк.
Реализует АПИ для получения и отправки информации о состоянии горного
перевала в разное время года. Предусмотрена возможность регистрации пользователя,
создание объектов в базе данных.
Код покрыт тестами для запуска тестов рекомендую использовать готовые фикстуры командой
python manage.py loaddata fixtures/model_name.json --app app.model_name
Используйте подключение всех установленных библиотек используйте команду
pip install -r requirements.txt
Для установки секретного ключа и информации для базы данных используйте
.env_example как пример
